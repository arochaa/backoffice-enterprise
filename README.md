## Start Project mode Developer

### Inicialize Project

Clone o projeto e insira as vareaveis de ambientes no arquivo .env para que o app funcione.

```bash
$ docker-compose down -v && docker-compose up
```

### Access Container

```bash
# Para acessar a maquina do serviço
$ docker exec -it backend-backoffice-enterprise bash
# Testes de conexão a base de dados de dentro da maquina de serviço
$ echo 'SELECT NAME FROM pis_sec.PIS_BO_USER pbu WHERE rownum <= 10;' | sqlplus -S pis_sec/PASSWORD@server-ip:server-port/server-base
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
